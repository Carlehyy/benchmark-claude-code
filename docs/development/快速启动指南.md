# 快速启动指南 - 5 分钟开始开发

## 前提条件

- ✅ 已安装微信开发者工具
- ✅ 已配置 Claude Code（本地部署，Opus 模型）
- ✅ 已安装 Node.js（v16+）

## Step 1：安装中文化 Claude Code 配置（2 分钟）

### 方式一：从 GitHub 安装

```bash
# 克隆您的中文化配置仓库
git clone https://github.com/Carlehyy/benchmark-claude-code.git

# 进入目录
cd benchmark-claude-code

# 安装到用户级配置（推荐）
mkdir -p ~/.claude
cp -r agents ~/.claude/
cp -r commands ~/.claude/
cp -r skills ~/.claude/
cp -r rules ~/.claude/
cp -r contexts ~/.claude/

# 验证安装
# 在 Claude Code 中输入 /help 查看所有命令
```

### 方式二：从本地文件安装

如果您已经下载了 `benchmark-claude-code-chinese.tar.gz`：

```bash
# 解压
tar -xzf benchmark-claude-code-chinese.tar.gz
cd benchmark-claude-code

# 安装
mkdir -p ~/.claude
cp -r agents ~/.claude/
cp -r commands ~/.claude/
cp -r skills ~/.claude/
cp -r rules ~/.claude/
cp -r contexts ~/.claude/
```

## Step 2：创建项目（1 分钟）

### 方式一：使用项目模板（推荐）

```bash
# 解压项目模板
tar -xzf logic-expression-miniapp-starter.tar.gz
cd logic-expression-miniapp

# 运行初始化脚本
bash setup-project.sh

# 安装依赖
npm install
```

### 方式二：手动创建

```bash
# 创建项目目录
mkdir logic-expression-miniapp
cd logic-expression-miniapp

# 下载并运行初始化脚本
# （从项目模板中复制 setup-project.sh）
bash setup-project.sh

# 安装依赖
npm install
```

## Step 3：配置微信小程序（1 分钟）

### 3.1 配置 AppID

编辑 `project.config.json`：

```json
{
  "appid": "your-appid-here",  // 替换为您的 AppID
  ...
}
```

如果没有 AppID，可以使用测试号：
1. 打开微信开发者工具
2. 选择"测试号"
3. 自动生成测试 AppID

### 3.2 打开项目

1. 打开微信开发者工具
2. 点击"导入项目"
3. 选择项目目录：`logic-expression-miniapp`
4. 点击"导入"

### 3.3 开通云开发（可选）

1. 在微信开发者工具中点击"云开发"
2. 开通云开发环境
3. 获取环境 ID
4. 编辑 `miniprogram/config/cloud.config.ts`：

```typescript
export const cloudConfig = {
  env: 'your-env-id',  // 替换为您的环境 ID
  traceUser: true
};
```

## 常见问题

### Q1：Claude Code 命令不工作？

**检查**：
- 是否正确安装了配置文件到 `~/.claude/`
- 在 Claude Code 中运行 `/help` 查看可用命令

### Q2：微信开发者工具编译失败？

**检查**：
- AppID 是否配置正确
- `project.config.json` 格式是否正确
- 是否安装了依赖：`npm install`

### Q3：云开发初始化失败？

**解决**：
- 确保已开通云开发
- 检查环境 ID 是否正确
- 查看微信开发者工具的云开发面板

### Q4：TypeScript 报错？

**解决**：
- 安装类型定义：`npm install --save-dev miniprogram-api-typings`
- 检查 `tsconfig.json` 配置
- 重启微信开发者工具
