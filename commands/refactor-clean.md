# 重构清理

通过测试验证，安全识别并删除无用代码：

1. 运行无用代码分析工具：
   - knip：查找未使用的导出和文件
   - depcheck：查找未使用的依赖
   - ts-prune：查找未使用的 TypeScript 导出

2. 在 .reports/dead-code-analysis.md 中生成综合报告

3. 按严重程度分类发现项：
   - 安全（SAFE）：测试文件、未使用的工具函数
   - 警告（CAUTION）：API 路由、组件
   - 危险（DANGER）：配置文件、主入口点

4. 仅建议安全删除项

5. 每次删除前：
   - 运行完整测试套件
   - 确认测试通过
   - 应用变更
   - 重新运行测试
   - 若测试失败则回滚

6. 显示已清理项的汇总

切勿在未运行测试的情况下删除代码！
