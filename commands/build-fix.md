# 构建与修复

逐步修复 TypeScript 和构建错误：

1. 运行构建：npm run build 或 pnpm build

2. 解析错误输出：
   - 按文件分组
   - 按严重程度排序

3. 对每个错误：
   - 显示错误上下文（前后各 5 行）
   - 解释问题原因
   - 提出修复方案
   - 应用修复
   - 重新运行构建
   - 验证错误是否已解决

4. 停止条件：
   - 修复引入了新的错误
   - 同一错误连续 3 次未解决
   - 用户请求暂停

5. 显示总结：
   - 已修复的错误
   - 剩余的错误
   - 新引入的错误

为安全起见，每次只修复一个错误！
