### 保留英文的内容

以下内容保留英文，原因如下：

1. **JavaScript 代码和注释**（scripts/ 目录）
   - 原因：代码注释保持英文是国际惯例，便于代码维护和国际协作

2. **详细指南**
   - the-shortform-guide.md
   - the-longform-guide.md
   - 原因：这两个文件较大（15-16KB），包含大量图片和外部链接，主要供高级用户参考

3. **配置文件中的技术字段**
   - 如 JSON 中的 name、tools、model 等字段名
   - 原因：这些是 Claude Code 系统要求的技术标识符，必须保持英文

4. **代码示例**
   - 所有 Markdown 文件中的代码块
   - 原因：代码本身应保持原样，不应翻译

## 使用说明

### 安装方式

#### 方式一：作为插件安装（推荐）

```bash
# 将此仓库添加为市场
/plugin marketplace add Carlehyy/benchmark-claude-code

# 安装插件
/plugin install everything-claude-code@benchmark-claude-code
```

#### 方式二：手动安装

```bash
# 克隆仓库
git clone https://github.com/Carlehyy/benchmark-claude-code.git

# 复制到 Claude 配置目录
cp -r benchmark-claude-code/agents/* ~/.claude/agents/
cp -r benchmark-claude-code/commands/* ~/.claude/commands/
cp -r benchmark-claude-code/skills/* ~/.claude/skills/
cp -r benchmark-claude-code/rules/* ~/.claude/rules/
```

### 快速开始

安装完成后，您可以立即使用以下命令：

- `/tdd` - 启动测试驱动开发
- `/plan` - 创建实施计划
- `/code-review` - 代码审查
- `/e2e` - 生成 E2E 测试
- `/refactor-clean` - 清理死代码
- `/setup-pm` - 配置包管理器
