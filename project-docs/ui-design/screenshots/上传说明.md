# 截图上传说明

## 📸 如何上传截图

### 方式一：通过 Git 上传（推荐）

```bash
# 1. 克隆仓库到本地
git clone -b document https://github.com/Carlehyy/benchmark-claude-code.git
cd benchmark-claude-code/project-docs/ui-design/screenshots

# 2. 将截图复制到对应目录
# 例如：将首页截图复制到 01-首页/ 目录

# 3. 提交到 Git
git add .
git commit -m "docs: 添加参考小程序截图"
git push origin document

# 4. 通知 AI 分析
# 在对话中说："我已经上传了截图到 ui-design/screenshots 目录"
```

### 方式二：直接在对话中发送

您也可以直接在对话中发送截图，我会帮您保存到对应目录。

## 📁 目录说明

```
screenshots/
├── 01-首页/              # 启动页、首页、导航等
├── 02-题库列表/          # 题库列表、筛选、搜索等
├── 03-题目详情/          # 题目展示、答案区域等
├── 04-答题界面/          # 答题交互、提交反馈等
├── 05-学习进度/          # 进度统计、数据可视化等

```

## 📝 截图命名建议

使用清晰的命名方式，便于识别：

```
[功能]-[状态].png

示例：
首页-默认状态.png
题库列表-筛选展开.png
题目详情-已答题.png
答题界面-提交成功.png
学习进度-周报.png
```

## ✅ 截图质量要求

- **清晰度**：截图清晰，文字可读
- **完整性**：包含完整页面，不要裁剪关键信息
- **状态**：尽量包含不同状态（正常、加载、空、错误）
- **尺寸**：保持原始尺寸，不要缩放
